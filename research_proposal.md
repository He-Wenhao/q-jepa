# 学习量子多体系统的最优压缩表示与迭代求解规则

## 1. 一句话总结

DFT = 人类选定密度作为latent space + 人类设计迭代规则（XC泛函）。我们的方法 = 从虚时演化轨迹中自动学习latent space + 自动学习迭代规则。DFT是本框架的一个特例。

## 2. 核心洞察：DFT-SCF迭代是虚时演化在latent space中的投影

量子多体系统求解基态有两条路：

**路径一：在全空间中做虚时演化。** 从某个初态 $|\psi_0\rangle$ 出发，沿虚时方向传播 $|\psi(\tau)\rangle = e^{-\tau H}|\psi_0\rangle / \|e^{-\tau H}|\psi_0\rangle\|$，激发态被指数压制，轨迹自然收敛到基态。这条路信息完整但计算成本随系统尺寸指数增长。

**路径二：在latent space中做迭代。** DFT选定电子密度 $\rho(\mathbf{r})$ 作为压缩表示（latent space），然后在密度空间中执行SCF迭代 $\rho_n \to \rho_{n+1}$，直到收敛到基态密度。每一步SCF迭代本质上是虚时演化在密度空间中的投影——同一个物理过程，只不过发生在压缩后的空间里。

DFT的巨大成功证明了路径二是可行的。但DFT有两个人为选择：

1. **Latent space的选择**：密度 $\rho(\mathbf{r})$。Hohenberg-Kohn定理保证这原则上足够，但密度丢弃了大量信息，导致精确泛函 $E[\rho]$ 极其复杂。在强关联体系中，现有的近似泛函严重失败。
2. **迭代规则的选择**：由近似的交换关联泛函决定。人类设计了几十年，仍未找到普适的精确形式。

**我们的提议**：不做这两个人为选择，而是从数据中同时学习最优的latent space和最优的迭代规则。

## 3. 方法：JEPA框架下的自监督学习

### 3.1 与LeCun JEPA框架的对应

JEPA的核心思想是：不在原始高维空间（像素）中做预测，而是学习一个latent space，在其中预测动态演化。

| JEPA | DFT（特例） | 本方法（推广） |
|---|---|---|
| 原始空间：像素 | 波函数 $\Psi$（指数复杂度） | 1-RDM $\gamma$（多项式复杂度） |
| Encoder：像素→latent | 固定映射：$\Psi \to \rho$（取对角/求迹） | 可学习的 Encoder：$\gamma \to z$ |
| Latent space | 密度 $\rho(\mathbf{r})$（人为选定） | $z$（数据驱动，待学习） |
| Predictor：$z_t \to z_{t+1}$ | SCF迭代：$\rho_n \to \rho_{n+1}$（近似泛函） | 可学习的 Predictor：$z_n \to z_{n+1}$ |
| 收敛到不动点 | 收敛到基态密度 | 收敛到基态latent $z^*$ |

### 3.2 蛋糕模型的对应

LeCun的蛋糕模型指出，智能系统的知识主要来自自监督学习（底层），监督学习和强化学习只是顶层的微调。当前计算化学的ML方法集中在中间层（监督学习：用DFT/CCSD标签训练神经网络势函数），底层的自监督学习几乎空白。

| 蛋糕层级 | LeCun框架 | 计算化学 |
|---|---|---|
| 底层：自监督（数据无限） | 视频帧预测 | 虚时演化的RDM轨迹预测 |
| 中层：监督（数据有限） | 图像分类 | 神经网络势函数、性质预测 |
| 顶层：强化学习（数据极少） | 决策/控制 | 实验数据驱动的修正 |

### 3.3 自监督信号：虚时演化的1-RDM轨迹

对给定的哈密顿量 $H$，从Fock态出发做虚时演化，在每一步提取1-RDM：

$$|\psi(\tau)\rangle = \frac{e^{-\tau H}|\psi_0\rangle}{\|e^{-\tau H}|\psi_0\rangle\|} \quad \longrightarrow \quad \gamma(\tau) = \text{Tr}_{N-1}[|\psi(\tau)\rangle\langle\psi(\tau)|]$$

轨迹 $\gamma(\tau_0), \gamma(\tau_1), \gamma(\tau_2), \dots$ 从某个初始1-RDM出发，逐步收敛到基态1-RDM。

这条轨迹的关键性质：

- **保证包含基态信息**：虚时演化的终点就是基态。
- **数据廉价**：不需要任何外部标签，轨迹自身提供监督信号。
- **物理上非平凡**：1-RDM的虚时演化方程依赖于2-RDM（BBGKY层级），因此predictor被迫隐式学习BBGKY闭合关系——这是一个数据驱动的多体近似。

### 3.4 模型架构

```
输入: 1-RDM γ(τ) + 哈密顿量参数 θ
        │
        ▼
   ┌──────────┐
   │  Encoder  │  ── γ(τ) → z(τ)，将1-RDM压缩为latent向量
   └──────────┘
        │
        ▼
   ┌───────────┐
   │ Predictor  │  ── z(τ) → ẑ(τ+δτ)，在latent space中预测下一步
   └───────────┘
        │
        ▼
   训练目标: ‖ẑ(τ+δτ) - sg(Encoder(γ(τ+δτ)))‖²

   其中 sg = stop-gradient，目标encoder用EMA更新，防止表示坍缩。
```

### 3.5 下游任务：从converged latent预测基态性质

训练完成后，对一个新的哈密顿量参数 $\theta$：

1. 从某个初始1-RDM $\gamma_0$ 出发
2. 编码为 $z_0 = \text{Encoder}(\gamma_0)$
3. 反复应用 Predictor：$z_0 \to z_1 \to z_2 \to \dots \to z^*$（收敛到不动点）
4. 在 $z^*$ 上接一个轻量 prediction head（MLP）：$z^* \to E_0$

Prediction head用少量基态能量标签训练。预训练的价值体现在：同样的标签数量下，有预训练的模型比从头训练的模型精度高得多。

此外，可以训练一个decoder $z^* \to \gamma_{\text{GS}}$ 来重建基态1-RDM，从而计算任意单体可观测量，验证latent没有丢失物理信息。

## 4. 第一篇论文：实验方案

### 4.1 实验平台

一维Hubbard模型：

$$H = -t \sum_{\langle i,j \rangle, \sigma} c_{i\sigma}^\dagger c_{j\sigma} + U \sum_i n_{i\uparrow} n_{i\downarrow}$$

参数范围：$U/t \in [0, 12]$，半填充，格点数 $L = 6, 8$。

选择理由：强关联程度可调，小尺寸可精确对角化，DFT在大 $U/t$ 下严重失败，正好是我们方法应该超越DFT的区间。

### 4.2 数据生成

**自监督数据（大量）**：

对每个 $U/t$ 值：
1. 精确对角化得到 $H$
2. 从多个随机Fock态出发做虚时演化
3. 在每个虚时步提取1-RDM，存储 $(\gamma(\tau), \gamma(\tau+\delta\tau))$ 配对

预计：~50个 $U/t$ 值 × 50个初始态 × ~50个虚时步 = ~125,000个自监督样本。

**监督数据（少量）**：

对每个 $U/t$ 值计算精确基态能量 $E_0$ 和基态1-RDM。人为限制用于训练的标签数量（20/50/100/200），模拟标签稀缺的真实场景。

### 4.3 基线对比

| 方法 | 描述 |
|---|---|
| **Baseline 1**：从头训练 | 同架构encoder + prediction head，无预训练，直接用基态标签训练 |
| **Baseline 2**：密度作为latent | 将1-RDM投影到密度（取对角元），再接prediction head |
| **Baseline 3**：完整1-RDM直接回归 | 不做压缩，直接从1-RDM用MLP预测能量 |
| **Baseline 4**：Hartree-Fock / DFT | 传统方法在Hubbard模型上的精度（尤其是强关联区间） |

### 4.4 核心实验与预期结果

**实验1：样本效率**

固定测试集，扫描训练标签数量（20, 50, 100, 200），画学习曲线。

预期：预训练模型在少标签（20-50个）时显著优于baseline 1，且优势在大 $U/t$（强关联）区间最明显。

**实验2：迭代收敛**

对新的 $U/t$ 值，从随机初始latent出发反复应用predictor，观察是否收敛到不动点，以及不动点对应的能量预测是否准确。

预期：模型学到的迭代在latent space中稳定收敛，类似SCF迭代的行为。

**实验3：latent分析**

计算学到的latent $z^*$ 与密度（1-RDM对角元）和完整1-RDM之间的互信息。

预期：在弱关联（小 $U/t$）区间，latent主要编码密度信息（与DFT一致）；在强关联（大 $U/t$）区间，latent保留了超越密度的1-RDM非对角信息。

**实验4：decoder重建**

训练decoder从 $z^* \to \gamma_{\text{GS}}$，检验重建的1-RDM的精度。

预期：重建精度高，证明latent未丢失关键物理信息。

### 4.5 评估指标

- 基态能量 MAE（分弱/中/强关联区间报告）
- 学习曲线（MAE vs 标签数量）
- Latent与密度/1-RDM的互信息（随 $U/t$ 变化）
- 迭代收敛速度与稳定性
- 1-RDM重建误差

## 5. 论文叙事结构

1. **引言**：DFT是计算化学最成功的方法，其本质是在人类选定的latent space（密度）中做迭代求解。但密度未必是最优的压缩表示，尤其对强关联体系。
2. **方法**：借鉴JEPA框架，提出从虚时演化轨迹中自监督学习latent space和迭代规则，DFT是本框架的特例。
3. **实验**：在Hubbard模型上验证，自监督预训练显著提升少标签下的基态预测精度，且学到的latent在强关联区间自动超越密度表示。
4. **讨论**：这一框架指向一个超越DFT的新范式——不预设描述变量，而是让数据决定最优的理论框架。
